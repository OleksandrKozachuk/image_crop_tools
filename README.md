# Custom image crop tool

## Installation

----

- `git clone https://github.com/OleksandrKozachuk/image_crop_tools.git`
- `cd image_crop_tools/`
- `python3 -m venv venv`
- `source venv/bin/activate (\venv\Scripts\activate - on Windows)`
- `pip install -r requirements.txt`
- `python run.py`

----
